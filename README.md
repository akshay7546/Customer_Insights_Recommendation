# Customer Insights and Recommendation System

## 📌 Project Description
Customer Insights and Recommendation System is a Data Science and Web-based project that analyzes customer purchasing behavior and provides valuable business insights through interactive dashboards and recommendation techniques.

The project includes customer segmentation, sales analysis, product recommendations, sentiment analysis, and business intelligence dashboards.

---

## 🚀 Features

- Customer Analysis Dashboard
- Product Analysis Dashboard
- Sales Analysis
- Customer Segmentation
- Product Recommendation System
- Sentiment Analysis
- Revenue Forecasting
- Power BI Dashboard
- MySQL Database Integration

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-link>
cd Customer_Insights_Recommendation
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
